
## 디렉토리 구조

```
/apis: api
/apis/request: request dto
/apis/resposne: response dto
/components/{화면명}: 해당 화면에서 사용하는 component
/pages: page에 해당하는 component
/stores: redux store
/stores/slices: redux store slice
/hooks: 커스텀 훅
/utils: 유틸 함수
/hocs: 고차 컴포넌트 (High order component)
/assets: 아이콘, 이미지, 오디오 파일 등등
```
